Ad Creation Guidelines

Layout Guidelines

  • Design digital pages to conform to the page area and page margin.
  • If possible, make the dimensions of the file to be the actual size that it will be printed at, with no margins.
  • Use the appropriate printing orientation(portrait or landscape) depending on the chosen advertisement dimensions.

Typography and Rules

To understand typographical requirements for a good quality print ad, please view our Typography Design Recommendations (pdf).

Color Space Requirements

  • Upload an ad created in CMYK color space for satisfactory image print quality.
  • The difference between color images created in RGB color space and CMYK color space is shown below:
  • How to create an ad in CMYK color space:
    • Illustrator: select 'Print' in the New Document Profile. This automatically sets the color space to CMYK.
    • Photoshop: select 'Print' in the Preset dropdown.
    • InDesign: select 'Print' in Intent dropdown. The document transparency blend space is set to CMYK.

Color Graphics and Image Files

  • Halftone images should be 200 dpi and line art images should be 600-1200 dpi. Prepare color images as CMYK.

Digital Image Capture

  • Use a minimum input resolution of 200 pixels per inch (ppi) for continuous tone images scanned at their final image size.

Note: Uploaded files which are not created in CMYK color mode will be automatically converted to CMYK color mode by the printing house and the newspaper agency is not liable for any color matching issues. Any files uploaded as CMYK will be printed the same.
